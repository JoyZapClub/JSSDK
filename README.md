# GameSDK Web Demo

A minimal browser demo for **GameSDK** (UMD build) showing:
- Login
- Portrait / Landscape fullscreen
- Exit WebView

## How to Use
1. Open `index.html` in a browser (preferably via a local server).  
2. Replace `params.gameId` with your actual Game ID.
